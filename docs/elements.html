Input modifiers

You can create a very minimalistic input by adding the underline class.

The invalid and disabled attributes work on all inputs, even textarea.

To disable an input add the disabled attribute:

To show that an input is invalid add the invalid attribute
adding invalid="false" attribute will make it normal again (visual only):

Or type a letter in the number input to trigger the webbrowsers built-in invalidation: 

<input type="text" class="underline" placeholder="Input with underline class" />
<input type="range" disabled />
<input type="number" disabled />
<input type="text" invalid />
<input type="text" invalid="false" />
<input type="button" value="submit" disabled>

Labels

Labels render with a font-size: 1.2rem; which results in 12px by default.
And have a font-weigth: 600;


Their presentation can be altered by a some classes or attributes:



If you add tabindex="0" as well as the required class, you get a tooltip which can be focussed:


You can modify the tooltip by providing data-tooltip="Your text"


You can also use the attribute disabled / disabled="true" or required / required="true" for the same effect of adding the class.
You can then set these to false to remove the effect.
disabled="false" and required="false" respectively. 

<label>A standard label</label>

<label class="required">Label with required class</label>
<label required>Label with required attribute</label>
<label required="false">Not required label</label>

<label class="disabled">Label with disabled class</label>
<label disabled>Label with disabled attribute</label>
<label disabled="false">Not disabled label</label>

<label class="required" tabindex="0">This is a required field with a tooltip</label>
<label required tabindex="0">This is a required field with a tooltip</label>
<label required tabindex="0" data-tooltip="Custom text">Required label with custom tooltip</label>
