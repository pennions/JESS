Main, Section, Fieldset

The elements <main>, <section> and <fieldset> elements are all displayed with display:flex.
The child elements are all displayed at width: 100% which results in a single column by default.
All columns will be automatically scaled down and wrapped when the viewport can not fit all the columns properly.
You can try it by resizing the browser.

N.B. The blue fill texture is for demonstration purpose only.

Single column

<main>, <section> and <fieldset> elements are 100% width.

All elements start out as width:100%; and will take up more space if there is room, up to their specified maximum.

Block 100% width
<section>
    <div>Block 100% width</div>
</section>

Two columns

  • It will display two columns when the viewport has min-width:576px;
