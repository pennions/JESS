Notification

A notification is a <dialog> that is directly inside a <body> and under a <nav> element to make it a top notification
and stick to the top when scrolling.
Or inside <body> and above the <footer> with the additional bottom class to make it appear above the footer and make it stick to the bottom when scrolling.
You can add or remove the open attribute to show / hide the notification.

Variants:

Default notification Warning notification Success notification

If you add a feather icon, it will have a contrasting color, matching the notification:

Error notification 
<dialog class="notification" open>
    <span>Default notification</span>
    <button class="close">X</button>
</dialog>

<dialog class="notification warning" open>
    <span>Warning notification</span>
    <button class="close">X</button>
</dialog>

<dialog class="notification success" open>
    <span>Success notification</span>
    <button class="close">X</button>
</dialog>

<dialog class="notification error" open>
    <svg class="icon mr-1">
      <use href="./icons/feather-sprite.svg#alert-triangle" />
    </svg>
    <span>Error notification</span> <button class="close">X</button>
</dialog>

Modal

A modal is a <dialog> with a class="modal".
It opens when it has a property open or open="true".
It closes when the property is removed or set to false.
You could use a card (article) for the modal as demonstrated below.

Message

I am a modal.

If you want to disable scroll on the body add the no-scroll class on the body.

Message

I am a modal.
 
<button
    onclick="document.getElementById('demo-modal').setAttribute('open', true)"
    >
    Open modal
</button>

<dialog
    id="demo-modal"
    class="modal"
    onclick="this.removeAttribute('open')"
    >
    <article class="normal w-25">
        <header>
            <h3>Message</h3>
        </header>
        <div>I am a modal.</div>
        <footer>
            <button
                onclick="document.getElementById('demo-modal').setAttribute('open', false)"
            >
                Close
            </button>
        </footer>
    </article>
</dialog>

<button
    onclick="document.getElementById('demo-modal-noscroll').setAttribute('open', true); 
             document.getElementsByTagName('body')[0].classList.add('no-scroll')"
    >
    Open modal with no-scroll applied.
</button>

<dialog
    id="demo-modal-noscroll"
    class="modal"
    onclick="this.removeAttribute('open'); 
             document.getElementsByTagName('body')[0].classList.remove('no-scroll')"
    >
    <article class="normal w-25">
        <header>
            <h3>Message</h3>
        </header>
        <div>I am a modal.</div>
        <footer>
            <button
                onclick="document.getElementById('demo-modal-noscroll').setAttribute('open', false); 
                         document.getElementsByTagName('body')[0].classList.remove('no-scroll')"
                >
                Close
            </button>
        </footer>
    </article>
</dialog>
N.B. You can also set a close on the outside of the modal. As done in the example. Try it!

Breadcrumb

A simple breadcrumb interface:

  • Home
  • Workshop
  • Sawbench
<nav aria-label="Breadcrumb">
  <ul>
    <li><a href="">Home</a></li>
    <li><a href="">Workshop</a></li>
    <li><a href="" aria-current="page">Sawbench</a></li>
  </ul>
</nav>
