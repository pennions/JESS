    ></span>
</label>

<label class="switch switch-margin">
    <input type="checkbox" hidden />
    <span class="toggle"></span>
    <span
        class="toggle-background label-left"
        data-label="Off"
        data-label-on="On"
    ></span>
</label>

<label class="switch">
    <input type="checkbox" hidden />
    <span class="toggle"></span>
    <span class="toggle-background" data-label="Off state only"></span>
</label>

<label class="switch">
    <input type="checkbox" hidden checked />
    <span class="toggle"></span>
    <span class="toggle-background" data-label-on="On state only"></span>
</label>
N.B. The data-label attribute is rendered as a pseudo element, make sure it will fit by applying the correct margins around the switch.
