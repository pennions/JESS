Typography

JESS uses the following font-size, based on a 62.5% of the default font-size.
This translates to 0.1rem equals 1px.

Element Specified font-size Computed pixels

h3 1.8rem 18px
h4 - 16px
h5 - 13.28px
label 1.4rem 14px
small - 13.3333px

Blockquote

Blockquote will have 100% width. To shorten it, use a width utility class. In this case w-16 is applied:

A famous quote of someone, spanning multiple lines. Anonymous

Blockquote inside an article, which has the no-padding class and a width utility class:

A famous quote of someone, spanning multiple lines. Anonymous 
<blockquote class="w-16">
    A famous quote of someone, spanning multiple lines.
    <cite>Anonymous</cite>
  </blockquote>

  <article class="no-padding w-16">
    <blockquote>
      A famous quote of someone, spanning multiple lines.
      <cite>Anonymous</cite>
    </blockquote>
  </article>
