Layout utility classes

Class Description
sticky Makes it stick to the top when scrolling down.
no-scroll Remove all scrollbars, handy to apply on the body when opening a modal.
overflow-horizontal Make an element crollable on the x-axis if the contents is too large.
no-stretch Apply this to a flex layout item like section, main or fieldset
to have its children keep their own size.
stretch Apply this to a flex layout item like section, main or fieldset
to have its children take up as much space as there is.
row Make an element display flex with the default row direction.
column Make an element display flex with the column direction
align-center Applies: align-items: center;
align-end Applies: align-items: flex-end;
align-start Applies: align-items: flex-start;
align-stretch Applies: align-items: stretch;
justify-center Applies: justify-content: center;
justify-around Applies: justify-content: space-around;
justify-evenly Applies: justify-content: space-evenly;
justify-end Applies: justify-content: flex-end;
justify-between Applies: justify-content: space-between;
justify-start Applies: justify-content: flex-start;
no-gap Sets the gap of a flex container to 0.
auto-width Makes an element take on an auto-width.
no-wrap Makes a text not wrap.
inline-block Applies: display: inline-block;
inline-flex Applies: display: inline-flex;

Padding / Margin utilities

Each version has a number ranging from 0 and 5 and auto. 0 means no padding or margin for that direction.

  1. is the default distance divided by 3 (so 1.5rem / 3 = 0.5rem)
  2. is the default distance - the distance of one (so 1.5rem - 0.5rem = 1rem)
  3. is the default distance
  4. is the default distance times one and a halve. (1.5rem * 1.5 = 2.25rem)
  5. is the default distance times two. (1.5rem * 2 = 3rem)
