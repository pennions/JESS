Labels

Labels render with a font-size: 1.2rem; which results in 12px by default.
And have a font-weigth: 600;


Their presentation can be altered by a some classes or attributes:



If you add tabindex="0" as well as the required class, you get a tooltip which can be focussed:


You can modify the tooltip by providing data-tooltip="Your text"


You can also use the attribute disabled / disabled="true" or required / required="true" for the same effect of adding the class.
You can then set these to false to remove the effect.
disabled="false" and required="false" respectively.
