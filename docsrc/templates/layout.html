Layout elements

The elements <main>, <section> and <fieldset> are all displayed with display:flex.
The child elements are all displayed by default at width: 100% which results in a single column.
All columns will be automatically scaled down and wrapped when the viewport can not fit all the columns properly.
You can try it by resizing the browser.

N.B. The blue fill texture is for demonstration purpose only.

Single column

All child elements start out as width:100%; and will take up more space if there is room, up to their specified maximum.

Center

Add class="center" to get a single element centered vertically and horizontally.

It has the following variants available:

  • class="center"
    • It will take 100% and will be centered horizontally.
    • if the screen is has min-width:576px; it will be centered vertically.
  • class="center full-height"
    • it will render with height:100vh;
  • class="center with-header"
    • It will subtract the height of the header from the area.
  • class="center with-footer"
    • It will subtract the height of the footer from the area.
  • class="center with-header with-footer"
    • It will subtract both height of the header and the footer from the area.
  • content-top
    • If you add content-top class to any of the above, the child will only be horizontally centered and not vertically
N.B. if you want the footer to stick to the bottom, apply a column class on the body and column-end class on the footer.

Aside layout

The aside layout can be used if you want to have a small - wide layout. horizontal layout.
The aside element has a width:auto;
The element on the right will have flex:1; and take the remaining space.
Only the first element must be <aside>, the right side can be any element. 

{{! aside_layout }}
Fun fact: the current page uses this aside layout for having a vertical navigation bar.

Nav

When you include a <nav> element directly inside a <body> element it will:

  • Have a box-shadow
  • Have max-height: 6rem;
  • Have padding: 1.5rem;
